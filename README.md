# Engly — Приложение для заучивания английских слов

## Описание

Веб-приложение для изучения английских слов. Пользователь видит слово на русском языке и должен составить его английский эквивалент из предложенных букв.

## Функционал

### Основной экран
- Отображение русского слова для перевода
- Набор букв для составления английского слова (включая лишние буквы)
- Визуальная обратная связь при правильном/неправильном ответе

### Система оценок
- Каждое слово имеет рейтинг (по умолчанию 0)
- Правильный ответ: +1 к рейтингу
- Неправильный ответ: -1 к рейтингу
- При рейтинге +5 и ошибке: сброс к 0
- В плюс уйти сложно, в минус — легко

### Выбор слов
- Случайный выбор слова с учётом рейтинга
- Слова с низким рейтингом выпадают чаще
- Слова с высоким рейтингом выпадают реже

### Статистика
- Количество слов, изученных сегодня
- Количество верных ответов
- Количество ошибок
- Общий процент «положительных» слов в словаре

### Настройки
- Добавление/редактирование словаря (формат: `русское слово - english word`)
- Сброс статистики
- Сброс всех данных

## Технологии

- **Tailwind CSS v4** — стилизация (CDN)
- **Alpine.js** — реактивность и логика (CDN)
- **Font Awesome** — иконки (CDN)
- **localStorage** — хранение слов и статистики
- **PWA** — работа оффлайн, установка на домашний экран

## PWA (Progressive Web App)

Приложение поддерживает:
- Работу без подключения к интернету
- Установку на домашний экран телефона
- Полноэкранный режим

### Установка на телефон
1. Открыть сайт в браузере (Chrome/Safari)
2. Нажать «Поделиться» → «На экран «Домой»»
3. Приложение будет работать оффлайн

## Хостинг

Сайт хостится на GitHub Pages (ветка `main`).

## Структура данных

### Слово
```json
{
  "russian": "слово",
  "english": "word",
  "rating": 0
}
```

```json
{
  "totalCorrect": 0,
  "totalWrong": 0,
  "todayCorrect": 0,
  "todayWrong": 0,
  "lastSessionDate": "2026-03-12"

```
