# Engly — Приложение для заучивания английских слов

## Описание

Веб-приложение для изучения английских слов. Пользователь видит слово на русском языке и должен составить его английский эквивалент из предложенных букв.

## Функционал

### Основной экран
- Отображение русского слова для перевода
- Набор букв для составления английского слова (включая лишние буквы)
- Визуальная обратная связь при правильном/неправильном ответе

### Система оценок
- Каждое слово имеет базовый рейтинг (из таблицы)
- Правильный ответ: +1 к рейтингу
- Неправильный ответ: -1 к рейтингу
- При рейтинге +5 и ошибке: сброс к 0
- В плюс уйти сложно, в минус — легко

### Выбор слов
- Случайный выбор слова с учётом рейтинга (с учётом локальных изменений)
- Слова с низким рейтингом выпадают чаще
- Слова с высоким рейтингом выпадают реже

### Словарь
- Просмотр всех слов с рейтингами (кнопка 📋 вверху)
- Сортировка: по русскому / английскому / рейтингу (с учётом локальных изменений)
- Цветовая индикация: зелёный = положительный, красный = отрицательный

### Статистика
- Количество слов, изученных сегодня
- Количество верных ответов
- Количество ошибок
- Общий процент «положительных» слов в словаре

### Синхронизация с Google Таблицей
- **Мгновенный старт** — приложение работает сразу из локального кэша
- **Фоновая загрузка** — словарь обновляется из таблицы после старта
- **Дельта изменений** — изменения рейтингов хранятся отдельно
- **Сохранение** — кнопка появляется при наличии несохранённых изменений
- **Оффлайн-работа** — полная функциональность без сети

## Технологии

- **Tailwind CSS v4** — стилизация (CDN)
- **Alpine.js** — реактивность и логика (CDN)
- **Font Awesome** — иконки (CDN)
- **localStorage** — раздельное хранение словаря и дельты
- **Google Apps Script** — синхронизация с Google Sheets
- **PWA** — работа оффлайн, установка на домашний экран

## PWA (Progressive Web App)

Приложение поддерживает:
- Работу без подключения к интернету
- Установку на домашний экран телефона
- Полноэкранный режим

### Установка на телефон
1. Открыть сайт в браузере (Chrome/Safari)
2. Нажать «Поделиться» → «На экран «Домой»»
3. Приложение будет работать оффлайн

## Настройка синхронизации с Google Таблицей

### Шаг 1: Создайте таблицу
1. Создайте новую Google Таблицу
2. Назовите первый лист `Sheet1` (или как угодно)

### Шаг 2: Установите Google Apps Script
1. В таблице: **Расширения → Apps Script**
2. Скопируйте содержимое файла `google-apps-script.js`
3. Вставьте в редактор Apps Script
4. Нажмите **Развертывание → Новое развертывание**
5. Тип: **Веб-приложение**
6. Доступ: **Все** (Anyone)
7. Нажмите **Развернуть**
8. Скопируйте URL веб-приложения

### Шаг 3: Заполните таблицу словами
В листе создайте заголовки и заполните данными:

| russian | english | rating |
|---------|---------|--------|
| кошка   | cat     | 0      |
| собака  | dog     | 0      |
| дом     | house   | 0      |

### Шаг 4: Работа с приложением
- При запуске приложение **мгновенно загружается** из локального кэша
- **Фоновая загрузка** обновляет словарь из таблицы
- После каждого ответа рейтинг обновляется локально (дельта)
- Появляется кнопка «Сохранить» вверху справа
- Нажмите «Сохранить» для отправки изменений в таблицу
- После сохранения дельта сбрасывается

## Архитектура синхронизации

```
┌─────────────────────────────────────────────────────────┐
│                    localStorage                         │
├─────────────────────┬───────────────────┬───────────────┤
│   engly_words       │   engly_delta     │  engly_stats  │
│   (базовый словарь) │  (изменения)      │  (статистика) │
│   [                 │  {                │  {            │
│     {rus, eng, rat} │   "кот|cat": 3,   │   correct: 5  │
│   ]                 │   "дом|house": -1 │   wrong: 2    │
│                     │  }                │  }            │
└─────────────────────┴───────────────────┴───────────────┘
         │                       │
         │  Фон: GET             │  Сохранение: POST
         │  (загрузка словаря)   │  (слова + дельта)
         ▼                       ▼
┌─────────────────────────────────────────────────────────┐
│                  Google Таблица                         │
│  (источник истины для базового словаря)                 │
└─────────────────────────────────────────────────────────┘
```

### Преимущества архитектуры

1. **Мгновенный старт** — не ждём сеть, работаем из кэша
2. **Оффлайн-работа** — полная функциональность без интернета
3. **Разделение данных** — словарь и изменения независимы
4. **Нет конфликтов** — дельта применяется поверх базового словаря
5. **Простота** — при сохранении отправляем объединённые данные

### Поток данных

```
1. Старт приложения:
   └─> Загрузка из localStorage (мгновенно)
   └─> Выбор слова (работаем!)
   └─> Фоновая загрузка словаря из таблицы

2. Изменение рейтинга:
   └─> Запись в engly_delta
   └─> Появление кнопки «Сохранить»

3. Сохранение:
   └─> Чтение engly_words + engly_delta
   └─> Объединение: rating = base + delta
   └─> POST в таблицу
   └─> GET подтверждение
   └─> Сброс engly_delta
```

## Структура данных

### Слово (базовое)
```json
{
  "russian": "слово",
  "english": "word",
  "rating": 0
}
```

### Дельта
```json
{
  "слово|word": 3,
  "дом|house": -2
}
```

### Статистика
```json
{
  "totalCorrect": 0,
  "totalWrong": 0,
  "todayCorrect": 0,
  "todayWrong": 0,
  "lastSessionDate": "2026-03-12"
}
```
