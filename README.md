# Engly — Приложение для заучивания английских слов

## Описание

Веб-приложение для изучения английских слов. Пользователь видит слово на русском языке и должен составить его английский эквивалент из предложенных букв.

## Функционал

### Основной экран
- Отображение русского слова для перевода
- Набор букв для составления английского слова (включая лишние буквы)
- Визуальная обратная связь при правильном/неправильном ответе

### Система оценок
- Каждое слово имеет рейтинг (по умолчанию 0)
- Правильный ответ: +1 к рейтингу
- Неправильный ответ: -1 к рейтингу
- При рейтинге +5 и ошибке: сброс к 0
- В плюс уйти сложно, в минус — легко

### Выбор слов
- Случайный выбор слова с учётом рейтинга
- Слова с низким рейтингом выпадают чаще
- Слова с высоким рейтингом выпадают реже

### Словарь
- Просмотр всех слов с рейтингами (кнопка 📋 вверху)
- Сортировка: по русскому / английскому / рейтингу
- Цветовая индикация: зелёный = положительный, красный = отрицательный

### Статистика
- Количество слов, изученных сегодня
- Количество верных ответов
- Количество ошибок
- Общий процент «положительных» слов в словаре

### Синхронизация с Google Таблицей
- **Автоматическая загрузка** из таблицы при старте приложения
- **Автосохранение** изменений рейтингов
- Индикатор несохранённых изменений (янтарная полоса внизу)
- Локальное кэширование для оффлайн-работы

## Технологии

- **Tailwind CSS v4** — стилизация (CDN)
- **Alpine.js** — реактивность и логика (CDN)
- **Font Awesome** — иконки (CDN)
- **localStorage** — локальное хранение
- **Google Apps Script** — синхронизация с Google Sheets
- **PWA** — работа оффлайн, установка на домашний экран

## PWA (Progressive Web App)

Приложение поддерживает:
- Работу без подключения к интернету
- Установку на домашний экран телефона
- Полноэкранный режим

### Установка на телефон
1. Открыть сайт в браузере (Chrome/Safari)
2. Нажать «Поделиться» → «На экран «Домой»»
3. Приложение будет работать оффлайн

## Настройка синхронизации с Google Таблицей

### Шаг 1: Создайте таблицу
1. Создайте новую Google Таблицу
2. Назовите первый лист `Sheet1` (или как угодно)

### Шаг 2: Установите Google Apps Script
1. В таблице: **Расширения → Apps Script**
2. Скопируйте содержимое файла `google-apps-script.js`
3. Вставьте в редактор Apps Script
4. Нажмите **Развертывание → Новое развертывание**
5. Тип: **Веб-приложение**
6. Доступ: **Все** (Anyone)
7. Нажмите **Развернуть**
8. Скопируйте URL веб-приложения

### Шаг 3: Заполните таблицу словами
В листе создайте заголовки и заполните данными:

| russian | english | rating |
|---------|---------|--------|
| кошка   | cat     | 0      |
| собака  | dog     | 0      |
| дом     | house   | 0      |

### Шаг 4: Работа с приложением
- При запуске приложение **автоматически загружает** данные из таблицы
- После каждого ответа рейтинг обновляется локально
- Появляется индикатор «Есть несохранённые изменения»
- Нажмите «Сохранить» для отправки изменений в таблицу
- После сохранения данные в таблице обновляются

## Архитектура синхронизации

```
┌─────────────────┐     GET      ┌─────────────────┐
│   Приложение    │ ───────────> │  Google Таблица │
│   (localStorage)│ <─────────── │  (источник)     │
└─────────────────┘     POST     └─────────────────┘
       │  ▲                        │
       │  └────────────────────────┘
       │       Обновления рейтингов
       └─ Загрузка при старте
```

1. **При старте**: GET → загрузка всех слов из таблицы
2. **При изменении**: локальное обновление + флаг `needsSync`
3. **По кнопке «Сохранить»**: POST → отправка всех слов в таблицу
